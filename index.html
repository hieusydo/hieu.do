PROJECTS

Imagely: Learn Words with Images

Built at hackNY 2015.


Imagely is a language learning tool that associate an image with seven words. Using Clarify's powerful image recognition system, the app retrieves words associated with the image url inputed by the user. It then uses Yandex translation API to translate those words into a language of the user's choice. Current supported languages are: Spanish, Russian, Vietnamese, Dutch and Hebrew. My team also think of making a quiz mode that will allow users to bookmark important words and review later.

Notr: Text Visualization in Mindmapping

Built at HackMIT 2015, my first hackathon in the US!


Notr is a text visualization software based on AngularJS with dynamic mind mapping capabilities. Mindmapping is awesome, but drawing one takes time and effort. Notr can render graphs based on input text with simple syntax and structure. It is a text based application that can parse the nodal relationships and render them on a graph. UI combines a text editor and a display for the graph. -@, #, -, and id. (Parser). -render graph with said relationships. This project is possible thanks to Ace.c9.io, angular-ui-ace, d3.js, ui-bootstrap

SIA: Talk About It - Sex-ed for Vietnamese Teens

Second Prize - Built at Hackademics Vietnam 2015, my first hackathon ever!


A sex education website designed to provide useful info and interactive data to equip teenagers with crucial knowledge about psychophysiological issues. These include but is not limited to safe sex, choice, and gender. The information is aggregated from trusted source and comes in various forms such as articles, charts, graphs, scenarios, et cetera. A sharing section also allows teens to post any concerns to receive advice, or simply to share their personal stories. This project is built based on: angular-google-chart and angular bootstrap.

Home Thermostat

First hardware project!


A simple, elegant home thermostat built with Arduino Uno. The temperature used in this project is TMP36, which is decent despite some inaccucracy. The sensor converts the change in voltage into temperature. The conversion process happens in the Arduino, an aweosome powerful microcontroller, which then displays the results onto the liquid display. It was a quick weekend project, and I had lots of fun doing it!
