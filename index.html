Alertify

Emergency Alert with Hand Gestures

An Android mobile and wear app that recognizes hand gestures to trigger emergency alert: Flick the fingers to send the current location, or twist the wrist call an emergency contact. Built with Java on Android Studio and Presto API.


Notr

Text Visualization for Mind-mapping

Notr is a text visualization software that auto-generates dynamic, interactive mind maps purely based on texts provided by users. Users only need to type in the text in a specific format, and the drawing will be done automatically. No more worries when you have good ideas but not-so-good drawing skills! Built with AngularJS, D3.js, and Ace.


Imagely

Active Language Learning with Images

Imagely is a language learning tool that associates an image with words. When users import an image, the app retrieves words associated with that image. Those words can be translated into Spanish, Russian, Dutch, Hebrew, and Vietnamese. A quiz mode will be added in the future. Built with AngularJS, Clarifai API, and Yandex API.


SIA

Sex-ed for Vietnamese Teens

A sex education platform that pioneers in promoting awareness among Vietnamese teens by encouraging them to share personal stories, and providing information aggregated from trusted sources. Built with AngularJS and Google Charts API.


Thermostat

My first hardware project!

My first hardware project! A simple, elegant home thermostat with a temperature sensor that converts the change in voltage into temperature. The result is shown on the liquid display. Built with Arduino Uno.
